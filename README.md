## About this repository

This program allows you to run one single script, and it will instantly remove OneDrive from your system. Additionally, it will prevent it from being installed in the future due to updates.


## Install

Simply navigate to the releases page in this repository, find the setup.exe file and run it. you may run into a Windows Defender message. ![WinDefender](assets/notsecure.png) Simply click More info, then run anyway. ![See more](assets/notsecuremore.png) Then click 'Yes' on the UAC prompt and follow the installer's instructions. Then, run the .bat file from the location you installed it to. Press any key. A UAC prompt will appear, click Yes, and then press any key again. The script will now remove OneDrive.
